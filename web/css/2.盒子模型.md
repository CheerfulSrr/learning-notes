# 边框(`border`)
## 属性
1. `border-width`: 宽度
2. `border-style`: 样式
3. `border-color`: 颜色
# 内边距
1. 边框与内容之间的距离
2. 增加内边距会撑大盒子尺寸
3. 如果未指定`width/height`, 内边距不会撑开盒子
4. 计算盒子内容的宽、高时, 应该减去内边距
## 属性
1. `padding-left`: 左内边距
2. `padding-right`: 右内边距
3. `padding-top`: 上内边距
4. `padding-bottom`: 下内边距
